Show example CSV format

Required columns:

Data SE,Infra SE,Apps SE,DSS (Growth/Acq),TPID,Customer Name,SME&C POD,Sales Territory,Vertical,Vertical Category,Primary Cloud & AI DSS,Primary Cloud & AI-Acq DSS,Primary Cloud & AI Apps DSE,Primary Cloud & AI Data DSE,Primary Cloud & AI Infrastructure DSE

Example row:

John Smith,John Doe,Busy Body,Mary Kay,12345678,BIG SALES CUSTOMER,East POD 06,East.SMECC.SDP.0603,IT Services & Business Advisory,Management Consulting,MARYKAY,,BUSYBOD,JOHNSMI,JOHNDOE

Column descriptions:

  • SME&C POD: Pod name (e.g., "Enterprise West")
  • Sales Territory: Territory name (e.g., "CA North")
  • DSS (Growth/Acq): Seller full name (determines seller)
  • Primary Cloud & AI DSS: Growth seller alias (leave empty if Acquisition)
  • Primary Cloud & AI-Acq DSS: Acquisition seller alias (leave empty if Growth)
  • Data SE: Data solution engineer name
  • Primary Cloud & AI Data DSE: Data SE alias
  • Infra SE: Infrastructure solution engineer name
  • Primary Cloud & AI Infrastructure DSE: Infra SE alias
  • Apps SE: Apps solution engineer name
  • Primary Cloud & AI Apps DSE: Apps SE alias
  • Customer Name: Full customer/account name
  • TPID: 7-9 digit Top Parent ID number
  • Vertical: Industry vertical (or "N/A")
  • Vertical Category: Industry category (or "N/A")
